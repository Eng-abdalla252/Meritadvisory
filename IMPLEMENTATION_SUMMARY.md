## 🎨 Visual Features

### Animations & Effects:

1. **Scroll Animations**
   - Elements fade in and slide up as you scroll
   - Staggered delays for sequential appearance
   - Smooth transitions (600ms duration)

2. **Hover Effects**
   - Cards lift up on hover (-translate-y)
   - Glow effects appear
   - Icons scale and change color
   - Shadows intensify

3. **Orbiting Icons** (Service detail pages)
   - 6 benefit icons orbit around central service icon
   - Floating animation (up and down)
   - Each icon has unique timing

4. **Pulsing Rings** (Differentiators section)
   - Animated rings pulse outward from central logo
   - Creates attention-grabbing effect
   - Continuous loop animation

5. **Color Coding**
   - Each differentiator has unique color
   - Consistent color scheme throughout
   - Gradient backgrounds for premium feel

### Responsive Design:

- **Mobile (< 640px)**: Single column, stacked layout
- **Tablet (640-1024px)**: 2-column grid
- **Desktop (> 1024px)**: 3-5 column grid
- **All breakpoints**: Smooth transitions between layouts
