## ✨ Animations

### On Scroll:
1. **Header** - Fade in + slide up (600ms)
2. **Stats cards** - Staggered fade in (100ms delays)
3. **Map section** - Fade in + slide up (1000ms delay)
4. **Country cards** - Staggered fade in (80ms delays)

### Continuous:
1. **Location pins** - Bounce animation (2s, infinite)
   - Staggered start (200ms per pin)
   - Moves up 10px at peak

### On Hover:
1. **Stats cards** - Lift up + glow appears
2. **Location pins** - Tooltip appears
3. **Country cards** - Lift up + border highlight

---

## 📱 Responsive Design

### Desktop (> 1024px):
- Stats: 4 columns
- Map + Countries: 2 columns side-by-side
- Map: Max width 448px (28rem)

### Tablet (768px - 1024px):
- Stats: 2 columns
- Map + Countries: Stacked
- Map: Centered

### Mobile (< 768px):
- Stats: 1 column
- Map + Countries: Stacked
- Map: Full width
- Country cards: 1 column

## 🎨 Technical Details

### Components Used:
- `Card` - For stats and country cards
- `MapPin` - For location markers

- SVG - For map outline

### Animations:

- Tailwind transitions for hover effects
- Staggered delays with inline styles

- Grid layouts (responsive)

- Shadow effects
- Border highlights
